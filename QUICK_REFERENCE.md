## 🏠 Home Screen

### What You Can Do
- ✍️ Write diary entry
- 😊 Select mood (5 options)
- 📁 Choose category (4 options)
- 📷 Attach photos
- 🎤 Record voice notes
- 💾 Save entry
- ✏️ Edit existing entry

### Important
- ❌ **NO DELETE OPTION** - You can only edit entries
- 📅 One entry per day
- 🔄 Auto-loads today's entry if exists

---

## 🔍 Search Screen

### 4 Search Modes

#### 1. Text Search
- Type keywords
- Searches entry content, mood, and category
- Real-time filtering

#### 2. Date Search
- Use ← → arrows to navigate dates
- View entry for specific date
- Shows exact date selected

#### 3. Month Search
- Browse by month and year
- View all entries for selected month
- Navigate with ← → arrows

#### 4. Category Search
- Filter by category
- Tap category button to filter
- Shows only matching entries

## ✏️ Edit Entry

### How to Edit
1. Go to Search or Timeline

### Remember
- ❌ **NO DELETE BUTTON** (by design)
- ✅ You can edit anytime
- ✅ Changes save immediately

## 📊 Data Storage

### What's Saved
- ✅ Entry text
- ✅ Mood selection
- ✅ Category selection
- ✅ Photos (as URIs)
- ✅ Voice recordings
- ✅ Date and time

### Where It's Saved
- 📱 Local device (AsyncStorage)
- 🔒 Private to your device
- 💾 Persists between sessions
- 🚫 No cloud sync (frontend only)
